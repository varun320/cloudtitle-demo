# Premium B2B / Proptech — Reference Findings

Bucket: "modern authoritative." Goal: borrow premium proptech confidence to elevate a Florida title agency without making it read as a tech startup.

---

## 1. Qualia — https://www.qualia.com

**Positioning:** "#1 rated closing software" for title and escrow. Direct adjacent (B2B SaaS *to* title agencies). Most useful reference because it speaks Cloud Title's own vocabulary.

**Visual treatment:** Navy + white, generous whitespace, contemporary sans-serif, moderate density. Hero is product-led ("Qualia Clear 2.0") with quantified value prop ("Automate 80% of title & escrow work"). Six-card product grid with iconography. Underwriter logo wall (Fidelity, First American) doing heavy social-proof lift. Compliance badges (SOC 2, ALTA-ALIGN) sit in-line, not buried in footer.

**What works:**
- Underwriter / partner logo wall as the *primary* trust mechanism — Cloud Title has the same lever (underwriter relationships, ALTA membership).
- Compliance badges treated as design elements, not afterthoughts.
- Quantified hero ("80% automated", named software version) instead of vague brand poetry.

**What doesn't translate:**
- It's product-led — Cloud Title sells a *service*, not a SaaS dashboard. Don't mimic the product-screenshot grid.
- Audience is title agencies *buying tools*; Cloud Title's audience is Realtors and buyers *hiring a closer*. Tone needs to be warmer.
- Illustrated landscape motifs skew startup; would feel off-brand against legal/trust positioning.

**Adopt:** Partial — palette discipline, underwriter wall, in-line compliance badges, quantified claims framed as "Based on internal averages."

---

## 2. Side — https://www.side.com

**Positioning:** White-label brokerage platform for top-producing agents. Closest tonal match: premium real-estate, audience overlap (Realtors).

**Visual treatment:** White ground, navy accent, confident sans-serif at display sizes, hero anchored by full-bleed video with overlay tagline ("Own Your Future"). Editorial/modular hybrid — data tiles ("500+ Companies", "5x transactions", "4–6 hrs saved") sit next to card-based agent success stories with real headshots. US-map graphic for geographic reach. Generous whitespace, almost no decorative chrome.

**What works:**
- Data tiles dropped into editorial flow — metrics get visual weight without becoming a dashboard.
- Real agent photography over stock — every face is a named partner. Authority through specificity.
- A single, declarative hero line carries the whole top fold.

**What doesn't translate:**
- Video-hero is heavy for Divi 5 and risky on mobile data; poster-frame fallback or skip.
- Side's audience is *agents only*; Cloud Title talks to agents *and* end consumers. Needs a second register for buyers.
- The "ambitious founder" emotional pitch is too aspirational for a closing transaction — buyers want competence, not destiny.

**Adopt:** Yes — editorial + data-tile hybrid, real Florida-team photography, single-line declarative hero, Florida-county map for service area.

---

## 3. Stripe — https://stripe.com

**Positioning:** Financial infrastructure. The reference for *how restraint signals premium* in B2B fintech. Borrow the grammar, not the look.

**Visual treatment:** White ground, subtle wave gradient, tight type system, bento grid sections, massive trust numerics ("US$1.9tn processed", "99.999% uptime", "500M+ API requests/day"). Sections alternate full-bleed imagery with constrained text columns — rhythm without ornament. Logo wall is Fortune-100 tier and unembarrassed.

**What works:**
- Quantified-trust framing — concrete numbers, not adjectives. Cloud Title equivalent: closings completed, average days-to-close, repeat-Realtor rate.
- Bento layout treats different content types (case study, metric, product card) as peers, breaking monotony.
- Restraint as a positioning tool — empty space *is* the luxury.

**What doesn't translate:**
- Stripe's gradient/wave aesthetic reads "developer tool"; we'd look like we're selling an API.
- Type voice is tuned for engineers — too cool for a title agency. Needs human warmth.
- Stripe earns minimalism with scale; a regional Florida agency needs more warmth to avoid feeling remote.

**Adopt:** Partial — bento structure, quantified-trust pattern, restraint discipline. Skip gradients and "infrastructure" vocabulary.

---

## Direction-specific takeaways

**Concept A — Editorial:** Lean Side. Hero = single declarative line + real Florida-team photography. Long-form sections (founder note, "how a closing actually works") with magazine-style pull quotes. Underwriter wall from Qualia. Skip Stripe entirely here.

**Concept B — Data-forward:** Lean Stripe + Qualia. Bento grid of stat tiles (closings, avg days, repeat-Realtor %, counties served), all labeled "Based on internal averages." Compliance badges in-line. Florida service-county map as a content tile, not decoration.

**Concept C — Premium minimal:** Lean Side + Stripe restraint. White ground, one accent extracted from cloudtitle.com palette, oversized confident type, one hero photograph, generous space, a single trust metric above the fold. Everything else earns its place or gets cut.

**Cross-concept rule:** Underwriter / ALTA / Florida-bar trust mechanisms must appear above the fold in all three — Qualia proves it works in this exact category.
